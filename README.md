# Continuous-Time Differential Games for LLM Alignment

Code and paper for "Continuous-Time Differential Games for LLM Alignment: Equilibrium Structure and Optimal Training Schedules."

## Structure

- `src/` — Core simulation code (LQ game, Riccati solver, Hopf analysis, PMP schedules)
- `experiments/` — Experiment scripts and configs
- `figures/` — Generated plots
- `paper/` — LaTeX source

## Setup

```bash
pip install -r requirements.txt
```

## Running experiments

```bash
python experiments/run_all.py
```
